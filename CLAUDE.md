# Glyf - Development Guide

- Tailwind v4 only — no inline styles, no component-scoped `<style>` blocks for
  layout/colour
- All colours via CSS variables: `bg-[var(--primary)]`,
  `text-[var(--foreground)]`
- Never hardcode hex values — use design tokens from `docs/globals.css`
- See `docs/style-guide.md` for component reference
- British english for everything
- 24 hour time format
